
## 一、k8s 常用命令
### 说明:以下命令，如果你没有加命名空间限制的话，默认就是在default下的

### 0、创建一个Nginx 应用  
    kubectl run nginx --image=nginx  --replicas=2

### 0.1 创建一个deployment 应用，传入环境变量
    kubectl run xej --image=spark:v1.0 --replicas=2 --env="ROLE=master"  --env="MASTER=192.168.1.110"
