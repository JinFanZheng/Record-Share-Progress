# calico常用命令

1. 删除profile策略 
    ```
    calicoctl delete profile cal_net1
    ```

2. 全互联模式设置   
    ```
    calicoctl config set nodeTonodeMesh off
    calicoctl config set nodeTonodeMesh on
    ```
